### 開發環境(Environment)

- Unity 版本：2022.3.14f1
- 使用 Visual Studio 作為 Scripts IDE（或其他）
- 其他相關工具：使用 Unity Hub 3.6.1 來編輯專案內容

### 怎麼玩這個遊戲(How to use play my game)

- 首先一進入遊戲會看到 Menu Scene
- 點擊 PLAY 就可以進入主遊戲
- 點擊 OPTION 就可以看到詳細操作角色的方法 
- 點擊 QUIT 就可以把遊戲關掉
- 使用 WASD 或 上下左右的箭頭 可以控制角色的移動(走路)
- 使用 LeftShift + WASD(上下左右箭頭) 可以實現跑步
- 按下 Z 可以實現旋轉攻擊
- 按下 Space 就可以實現跳躍
- 按下 F 可以撿起打敗敵人的金幣
- 按下 O 可以跳出 Menu Scene
- 當角色血條歸零就會跳出End Game Scene 點擊 Goto Menu 就可以回到 Menu Scene

### 加分事項(Bonus)

-進入主遊戲後會有音樂
